## 🚀 Getting started

Clone the project and access the folder

```bash
$ git clone https://github.com/C0bal/notify-me && cd notify-me
```

Use the [npm](https://www.npmjs.com/) to install dependences, follow the steps below.

```bash
# Install the dependencies
$ npm

# Start the project
$ npm start
```
